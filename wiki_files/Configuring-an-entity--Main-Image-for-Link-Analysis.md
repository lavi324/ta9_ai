## _This capability displays a custom image for an entity, within the Link Analysis skin._


**Requirements:**
1. The entity definition's Field Role must be equal to 5 (Main Image).
2. The entity definition's Data Type must equal 'Image'.
3. The image size must not exceed 40MB.

**Process**:
1. Create or modify the relevant entity definition, and add a new Entity Property Definition:
![image.png](/.attachments/image-7bb2bb37-9c88-41d8-a206-9a41536c934a.png)

2. In the Entity Property Definition Window, set Role=Main Image and Data Type=Image:
![image.png](/.attachments/image-888ebe72-feb9-4da6-b96e-663e8f339218.png)

3. Add the Entity Property Definition to the Layout, click OK, and save the Entity Definition:
![image.png](/.attachments/image-106102b4-12ff-47b3-8c48-0a82a6ceca8d.png)
![image.png](/.attachments/image-6d90d3de-0de3-42a0-946d-e391bd148ae2.png)
![image.png](/.attachments/image-e2da6b2c-b6f4-4967-b241-9efc1892b59a.png)

4. Create a new entity in the UI, and upload an image:
![image.png](/.attachments/image-879815c4-7d39-48bb-99b1-6eea00c46f3c.png)

5. The image will now be displayed in the Link Analysis skin:
![image.png](/.attachments/image-fedc4958-3122-4004-90e6-4caacf37229a.png)

_*Notice that zooming out too much will cause the image to disappear._*
