# Online User Guide

## Add / Manage Online users
TA9 Online user guide is password protected.
To allow clients to access our guide - they should be granted a dedicated user and password, allowing them to see the user guide page.




To add a new user to the user guide, follow the steps below:

### Step 1 - Log in to TA9 website Dashboard as admin user
- Open the Address https://www.t-a9.com/ta9-dashlogin/ 
- Login with Admin User credentials
- Navigate to the "users" tab and see the following screen
![image.png](/.attachments/image-717146a3-26dc-45f1-8e7c-ebf597f41ac8.png)

---

### Step 2 - Create a new user
- From the "users" tab - click on "add new"![image.png](/.attachments/image-965c1c16-023e-4121-988d-dc0d20b996d7.png)

- Fill in the user details as follows:
= User Name & Email - Must
= Password - generated by you, will use the client to login
= Check the confirmation of a weak password if the password you generated isn't strong 
= Check the "send email" to the user to notify them of the new account
= Set role to "markdown user" - note that it will be changed later
= Click on "add new user" 
![image.png](/.attachments/image-2e975ceb-438f-4d90-9176-3439dc55621a.png)
<BR>
Once the form is completed - the new user should exist in the database:
![image.png](/.attachments/image-a1162b0d-05d7-4e8d-9f17-667a747c134b.png)

>Note: Every Email can only be registered once.
---

### Step 3 - Change the User's role
- Locate the user in the user's table and click on it to enter the "edit" page
- Uncheck the "show toolbar" checkbox ![image.png](/.attachments/image-da01d7dc-aecd-474f-9972-ceb341f351b9.png)
- Scroll down and locate the "role" parameter, and change it to "no role for this site"
![image.png](/.attachments/image-a6c5311a-7b12-4783-904c-8268ce5e9c7c.png)
- Save the user details

---

### Step 4 - Test
Open the TA9 website and test the user you created and validate it can log in to the user guide page.

---

## Reset password
Open the user from the "users" page, and locate the "set new password" under the "accounts" section:
![image.png](/.attachments/image-89c8351f-9dbe-4f19-9b94-fdbb8ec0ac69.png)

Choose a new password and save.
>Note: "Send email to user" feature isn't working.
