#Pfsense
Pfsense is used as a DNS resolver 
1. Go to 10.100.102.2 and login
2. From the main menu go to Services --> DNS Resolver

   ![image.png](/.attachments/image-fa2d74a3-dee8-4d2b-a2c4-0a5559d57933.png)
3. to add a new host scroll down and press "Add" under Host Override.
   ![image.png](/.attachments/image-cae4836e-8b72-4422-a358-2d1c52698810.png)
   * Don't forget to apply the changes made.
