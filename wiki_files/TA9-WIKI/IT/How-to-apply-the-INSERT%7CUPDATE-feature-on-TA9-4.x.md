#User Configuration
1. To apply the INSERT|UPDATE feature, first you need to give permission to a user that will be able to do it
2. Go to the Admin Studio
3. Click on users and go to the Actions tab
4. click on the 'Insert DM' and the 'Update DM' checkboxes like the image below
![image.png](/.attachments/image-c167defe-034e-4b5f-a49a-6d82acf95888.png)
5. Click Save

#DM Configuration
1. In Admin Studio click on 'Data Models'
2. The DM must have one 'is ID' field 
3. In the DM page click on 'Repository'
![image.png](/.attachments/image-5253293c-4ac2-413e-aa98-aba685d18b8c.png)
4. Now checkbox the 'AllowInsert' and 'AllowUpdate' rows
![image.png](/.attachments/image-97cb6a1d-3b8b-45d2-8c57-8bff35eb7986.png)
6.make sure the 'is managed' checkbox is checked and permission mode is none
![image.png](/.attachments/image-605f1e25-d97e-41a8-8554-a2b625b638d4.png)
6. Click Save

#Applying the feature
1. Go to your DM in the TA9 UI
2. This is what the DM should look like
![image.png](/.attachments/image-f75029f0-4148-490e-aaea-2973990fd126.png)
3. To insert a new row to the DM, click on the '+' sign marked in red in the image above
4. Your page should look like this now with your fields
![image.png](/.attachments/image-8cadc861-be22-4188-9c7d-0c2cfc19c330.png)
5. Enter the values you want and click on save at the bottom right to create the new row in the DM
6. To edit a row in the DM click on Edit, the page should look like this with your fields
![image.png](/.attachments/image-07d27f02-bfc0-4a80-a09c-a96783688e1b.png)
