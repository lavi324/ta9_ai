1. Go to the DM you want to create a gantt skin for in Admin Studio 
2. Click on 'Set Gantt Templates'
3. Click on 'New', It should open a pop up like this

![image.png](/.attachments/image-2982c60d-0983-4701-baec-a9085f389cc7.png)

##gantt template parameters explanation:
A. Data Model - DM ID (not editable)
B. Template Name - Enter a name for the template itself (Mandatory)
C. Template Label - Select a Template label from the DM columns (Will display the value of the column chosen as label)
D. Start Date - Select a date column to mark the beginning (Mandatory)
E. End Date - Select a date column to mark the ending (Optional, if chosen cannot choose duration)
F. Duration Units - Select a unit of measure for the duration (Optional, Mandatory if chosen duration)
G. Duration - Select duration (Optional, if chosen cannot choose end date)
H. Status - Select a column as a status (Optional)
I. Color - Select a column which has a lookup to define colors for the labels (Optional)
J. Tooltips - Select columns that will display when you hover over the label (Optional)
K. Additional columns - Select more columns that will be displayed in the grid next to the gantt (Optional)
L. Hierarchies - select the order of the columns you want to be displayed in the grid next to the gantt (Optional)
