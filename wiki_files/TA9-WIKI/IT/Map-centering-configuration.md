## Goal
To change the default center of the map in a specific environment in the TA9 web system

## Steps
1. Open the TA9 web system
2. Click on Admin Tools
3. Clik on System Configuration
4. Select the 'web client' module
![image.png](/.attachments/image-9313f951-bbf3-47d0-97a4-fbb740077986.png)
5. Search for the 'mapOptions_center_lat' & 'mapOptions_center_lon'
6. Click on Edit 
![image.png](/.attachments/image-9f31dc48-6bd6-4ae6-a78f-1ef27ebb8839.png)
7. Change the config Value to the center of the environment's relevant country
![image.png](/.attachments/image-a7b9e934-68c7-4be7-af77-5b221b260781.png)

8. Cleare cache and hard reload the system, to check your new configuration.
