# 1. Sign to Ta9 system with any user of your choice.
# 2. click on the `+` button 
# 3. Click on `Add Entity` button

![image.png](/.attachments/image-396a0361-f341-42e0-9d65-25a249fd10d8.png)

# 4. Choose person entity.
![image.png](/.attachments/image-1c8fec85-49c7-4bf2-8b1a-2e8fa26dbc33.png)

# 5. Fill out all entity details, make sure you are fill out all the required fields.

# 6. Click on the 'save' button

![image.png](/.attachments/image-ccc18e46-391b-41e0-910e-3b6cfa81e59a.png)

# 7. Wait 15 minutes

# 8. Open the 'Federated Search' and search for the entity you have just created.

# 9. If the entity appears, then the indexing process completed successfully!
