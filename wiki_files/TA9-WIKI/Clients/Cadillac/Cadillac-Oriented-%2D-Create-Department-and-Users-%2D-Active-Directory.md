**3. Insert department name**

**4. Connect an Existing User to a Department (Organizational Unit):**


Right-click on the requested Department you want to connect the user to and choose: 'New' > 'Contact'

![image.png](/.attachments/image-c2b86e59-f24b-4a23-be0d-952bcf755f67.png)

- Fill in first and last name 
- Fill in the Full name =  login name + domain 
- Click 'ok':

![image.png](/.attachments/image-b715b4d7-d5c3-48b8-82ad-e307deedd861.png)
