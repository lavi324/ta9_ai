##Introduction
When we would like to see a column date in DM to the HH:MM:SS format, we can easily set the definition from the Admin Studio.

## Column definitions in the 'Admin Studio'
1. Open the Admin Studio module
2. Navigate to the Data Model you would like to change column display
3. Click on the column name
4. Change the "View" to be "Time Viewer Template"
![image.png](/.attachments/image-16070809-d604-4f90-a65b-010c215877ca.png)
5. Save the data model
6. Clear Server and Client Cache
