##Introduction
The "Lookup Manager" app helps us manage the lookups values, we can add, edit and delete values.

## How to use the lookup manager app?
1. Navigate to the "Apps" from the navigation bar
2. Open the "Lookup Manager" app
3. Search for the lookup you would like to update and click on it
![image.png](/.attachments/image-220ad2c2-ea9b-41a9-86d4-59bb0752a2c3.png)
4. You can click on the "Add" button to add a new value to the lookup
![image.png](/.attachments/image-853570cb-edd6-4743-94b8-29e9b971e251.png)
5. Click on the "Edit" button to edit the lookup value
6. Click on the "Delete" button to delete a lookup value 
![image.png](/.attachments/image-2e27197c-3f83-4513-8a99-280f80eaa6b3.png)
