**General details**
In order to connect with the client's email service provider (Zimbra), TA9 must connect with the SMTP server available on the Evoir.net domain.

Rayzone had provided TA9 with a new mailBox: Ta9@evoir.net
This proxy is created in order to allow sending external emails.
Password for the ta9 account is kept in the all machines password file.

The server that will be sending the emails is: winsrv02

**sending emails configuration:**

SSL: True
Port: 587
