# About

This module contains the login application.

## Core Components

### Login
Main login page

![image.png](/.attachments/image-c163cd94-eb71-4a39-8c32-20377036dd50.png)

The first screen the user encounter when getting to the analyst application.

### Verify
![image.png](/.attachments/image-4248ef11-9d4b-4a23-bf1b-a572be298188.png)

If the user that tries to login is 2FA and logged in in the past - this screen let him/her enter a 2FA code in order to login.

### Qr-code

![image.png](/.attachments/image-5b5f60bd-47d2-4667-a45d-7a0092e74353.png)
In this screen, using qr application, a 2FA user shall scan a qr code if it is his first time logging in. After scanning it - he will get a QR-CODE that changes every few seconds. This code shall be filled in the text box in order to successfully log in.

### Change password

In this page the user is required to change his/her password.
