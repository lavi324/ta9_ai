# This section is a knowledge base containing information about TA9.NG Angular project.

## Project General Guild Lines

- Eventually migrate all TA9.app AngularJS Project to the latest Angular 8 framework (now days the framework we use is Angular 8).
- Shall contain all new Angular developments.
- All modules will be written in [Redux](https://redux.js.org/introduction/getting-started) design pattern when applicable.
- We use [NGRX](https://ngrx.io/) as our Redux framework as our state management framework.
- You can find the project files [here](https://dev.azure.com/ta-9/_git/Argus?path=%2FWeb%2FTA9.NG&version=GBdev&_a=contents).
