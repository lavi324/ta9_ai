#TA9 Process
Custom Process for Azure DevOps boards, inherited from Agile. to customize this please follow the [Documentation](https://docs.microsoft.com/en-us/azure/devops/organizations/settings/work/manage-process?view=azure-devops)
Usually the flow should be like this:
* New -> Active -> Fixed -> Code Review -> QA -> Resolved -> Closed.
