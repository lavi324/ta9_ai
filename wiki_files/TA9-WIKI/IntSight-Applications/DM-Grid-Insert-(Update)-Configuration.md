In this new feature, the user has the option to insert (update) data to the Grid skin of a data model through the application GUI.

Simply by clicking the + button on the top right of the page, the insertion window will open: 

![image.png](/.attachments/image-66a41245-22f1-465c-8bf8-fafc7375e9a5.png)

![image.png](/.attachments/image-3ca45e77-f9b7-4d9e-9868-18c7a9667bed.png)


Click save after adding the data and the data model will be updated with the new data. 


**Prerequisites:** 

* The user must be permitted to have access to the data model in the admin studio:
![image.png](/.attachments/image-5ab36a56-b00b-4f54-9bcc-a9fa73127fd6.png) 

* The user must be permitted with the action of adding data into DM through the admin studio:
![image.png](/.attachments/image-f1681036-3488-407b-9bd0-dca1140fb308.png)
 
* The “Event Dashboard” DM must be set as “edible DM” through the admin studio.
