# General
> ℹ️ A dashboard is a cluster of widgets the user can create for the use of his organization.
 Using Data Models can be an easy way to view and search data from multiple sources on the same page.

# Prerequisites
- The Data Models must not be empty
- The user must define the widget settings and save them to view the Data Model

# How to create a Dashboard containing Data Models


_Step 1:_

Enter IntSight's Apps feature and click the "Dashboards" app:

![Image Linking](https://ta9test.files.wordpress.com/2023/04/dc-1.jpg)

_Step 2:_

Click "Create Dashboard":

![Image Linking](https://ta9test.files.wordpress.com/2023/04/dc-2.jpg)

_Step 3:_

Click "Add New Widget" and then click "Data Model":

![Image Linking](https://ta9test.files.wordpress.com/2023/04/dc-3.jpg)

_Step 4:_

Click the cog-wheel icon and set up your desired Data Model. Repeat the process to set all the Data Models you desire.

![Image Linking](https://ta9test.files.wordpress.com/2023/04/dc-4.jpg)

![Image Linking](https://ta9test.files.wordpress.com/2023/04/dc-5.jpg)

_Step 5:_

View the Data Models. Note that it may take some time for the data the load.

![Image Linking](https://ta9test.files.wordpress.com/2023/04/dc-6.jpg)
