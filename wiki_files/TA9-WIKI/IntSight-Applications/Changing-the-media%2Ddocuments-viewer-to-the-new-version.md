#Introduction
The new viewer enables the user to view the different files (documents and media) in a side viewer in a convenient way. When deploying a new environment we might want to change the old viewer to the new viewer.


# Configurations In The Client
Follow the following steps:
1. Log-in with an admin user
2. Navigate to the 'Admin Tools'
![image.png](/.attachments/image-b7794a64-c2d5-44de-8ee1-4315903189d0.png)
3. Click on the 'System Configurations' tile
![image.png](/.attachments/image-e4280a2b-910e-4759-8af8-c541515ee9f7.png)
4. Click on the 'Select Module' field and search for 'Web Client' in the search box
![choosing_web_client.gif](/.attachments/choosing_web_client-69a83997-f48a-47e7-be09-6e291b030897.gif)
5. Search for the "isDisplayNGAppViewer" config key![choosing_config_key.gif](/.attachments/choosing_config_key-d0625b3b-7056-495c-8b2b-7966eae6ee19.gif)
6. Change the 'Config Value' from 'False' to 'True'
![image.png](/.attachments/image-58497478-36b3-497c-b4ec-15e43c4251f5.png)
