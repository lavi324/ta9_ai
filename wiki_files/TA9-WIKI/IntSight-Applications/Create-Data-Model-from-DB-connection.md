1. Open admin studio.
2. Click on data models

![image.png](/.attachments/image-5299a37c-6638-4171-ac24-a97605d9498a.png)

3. Click on Add a data connection 
![image.png](/.attachments/image-f0be4dd3-41ea-4d87-b0e0-2d6919f2b499.png)

4. Choose your DB 

![image.png](/.attachments/image-053dcb99-6721-4bb1-9599-6ac8bcc157cf.png)

5. Add the credentials to your DB and click OK (You can click on test connection to verify your credentials are correct).  

![image.png](/.attachments/image-37a37178-81f7-4bd3-9e5a-8dc125db8bef.png)

6. Choose the Schema and data table from which you want to create the data model.
and check the main window checkbox or category that the data model will appear in the system.

![image.png](/.attachments/image-70b15a98-c4b5-426a-b263-1cae24f95275.png)

7. Click on save and the data model will be created.
