Copy the "TA9 Admin Client" folder (C:\Program Files\TA9\C#) from the TA9 server machine locally, and replace the server IP in the following configuration files: 

- [ ] TAStudio.exe.config 

- [ ] App.config 

![image.png](/.attachments/image-ed6188bf-0e8e-4906-9950-912a6799f68e.png)

Make sure that the server IP is changed: 
For example:

![image.png](/.attachments/image-37e7d909-1484-4137-b959-ed80277a30b6.png)

After the operation, save both files and open the admin studio from the exe file: 

![image.png](/.attachments/image-b9413a9a-3b05-4f9a-acaa-4981dee53cd9.png)

- [ ] your user name + domain
